## Core Functions

### Data Visualization Functions

| Method | Description |
|--------|-------------|
| [`log`](/visualizations#log) | Display log messages with timestamps |
| [`table`](/visualizations#tables) | Display tabular data |
| [`json`](/visualizations#json) | Display JSON data with interactive explorer |
| [`markdown`](/visualizations#markdown) | Render markdown content |
| [`progress`](/visualizations#progress) | Show progress bars (0.0 to 1.0) |
| [`number`](/visualizations#numbers) | Display numbers with formatting |
| [`card`](/visualizations#cards) | Display card-style information |

### Chart Functions

| Method | Description |
|--------|-------------|
| [`bar`](/visualizations#bar) | Create bar charts |
| [`area`](/visualizations#area) | Create area charts |
| [`pie`](/visualizations#pie) | Create pie charts |

### Utility Functions

| Method | Description |
|--------|-------------|
| [`location`](/visualizations#location) | Display location data on maps |
| [`raw`](/visualizations#raw-data) | Display raw data without formatting |
| [`stack`](/visualizations#stack) | Display current call stack with local variables |
| [`clear`](/advanced#clearing-data) | Clear all visualizations |
| [`wait`](/advanced#waiting-until-data-is-viewed) | Wait for data to be displayed (useful for scripts) |
| [`Shellviz`](/server) | Initialize Shellviz server manually |
| [`show_qr_code`](/advanced#using-your-phone-as-a-second-screen) | Display the connection string as a QR code |
