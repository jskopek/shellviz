<VizHeading slug="tables">Tables</VizHeading>

JSON data can be examined in two ways: as a color-coded, collapsible view, and as a series of cards that can be searched or filtered.

<CodeRunner code="shellviz.table([['Name', 'Age'], ['Rafael', 30], ['Charlie', 53]]);">
  <CodeExample language="python" code={`from shellviz import table
table([['Name', 'Age'], ['Rafael', 30], ['Charlie', 53]])
table([{'name': 'Rafael', 'age': 30}, {'name': 'Charlie', 'age': 53}])`} />
  <CodeExample language="javascript" code={`import { table } from 'shellviz';
table([['Name', 'Age'], ['Rafael', 30], ['Charlie', 53]]);
table([{name: 'Rafael', age: 30}, {name: 'Charlie', age: 53}]);`} />
</CodeRunner>

<VizHeading slug="progress">Progress</VizHeading>

Show a progress bar displaying 0-100% value. Can parse a wide variety of text, including strings, integers, and 0.0-1.0 floats.

<CodeRunner code="shellviz.progress(0.75)">
  <CodeExample language="python" code={`from shellviz import progress
progress(30)
progress("45%")
progress(.9)`} />
  <CodeExample language="javascript" code={`import { progress } from 'shellviz';
progress(30);
progress("45%");
progress(0.9);`} />
</CodeRunner>
